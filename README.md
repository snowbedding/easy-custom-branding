## ğŸ“– Usage

After installation, navigate to the settings page at **Settings â†’ Easy Custom Branding**. The settings are organized into four tabs:

-   **Login Page:** All options related to the `/wp-login.php` page.
-   **Admin Area:** Options for customizing the WordPress dashboard and admin pages.
-   **Email Settings:** Configure the sender information for emails sent by WordPress.
