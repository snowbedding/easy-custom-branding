A complete solution to customize your WordPress branding in the admin area and the login page.

== Description ==

Easy Custom Branding provides a comprehensive set of tools to replace the default WordPress branding with your own. It's the perfect plugin for developers and agencies who want to offer a consistent, white-label experience to their clients. Customize everything from the login page logo to the admin footer text, all from a simple settings page.

== Features ==

*   **Login Page Customization:** Change the logo, background image, form colors, and add custom CSS or HTML.
*   **Admin Area Branding:** Set a custom admin bar logo and footer text.
*   **Hide WordPress Elements:** Remove the WordPress version number, hide dashboard widgets, and hide admin menu items for non-admin users.
*   **UI Cleanup:** Hide the 'Help' and 'Screen Options' tabs for a cleaner interface.
*   **Custom Email Sender:** Change the default 'From' name and email address for all emails sent by WordPress.
*   **General Tweaks:** Hide the admin bar on the frontend and clean up unnecessary tags from your site's header.

== Installation ==

1.  Upload the `easy-custom-branding` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Go to **Settings → Easy Custom Branding** to configure the plugin.

== Frequently Asked Questions ==

= Where do I configure the settings? =

All settings can be found in your WordPress dashboard under **Settings → Easy Custom Branding**.

= Can I hide menu items from specific users? =

The plugin allows you to hide admin menu items from all users who are not administrators.

= Will my custom CSS be removed when the plugin is updated? =

No, all your settings, including custom CSS, are stored in the database and will not be affected by plugin updates.
